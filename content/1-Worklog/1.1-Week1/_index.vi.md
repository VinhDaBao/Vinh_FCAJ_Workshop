### Thành tựu của tuần 1:

* Hiểu được các khái niệm nền tảng của điện toán đám mây và lợi ích của việc sử dụng dịch vụ đám mây so với hạ tầng truyền thống.

* Tìm hiểu hạ tầng toàn cầu của AWS, bao gồm:
  * Regions
  * Availability Zones
  * Edge Locations
  * Global Services

* Tạo và bảo mật thành công tài khoản AWS Free Tier để thực hành.

* Làm quen với AWS Management Console và hiểu cách các dịch vụ AWS được tổ chức.

* Tìm hiểu mục đích của AWS Identity and Access Management (IAM), bao gồm:
  * IAM Users
  * IAM Groups
  * IAM Policies
  * Nguyên tắc Least Privilege

* Cài đặt và cấu hình thành công AWS CLI, bao gồm:
  * Access Key ID
  * Secret Access Key
  * Default Region
  * Output Format

* Sử dụng AWS CLI để thực hiện các tác vụ quản trị cơ bản như:
  * Xác minh cấu hình CLI
  * Truy xuất các vùng AWS có sẵn
  * Liệt kê IAM users
  * Xem cấu hình tài khoản
  * Khám phá các dịch vụ AWS có sẵn

* Hiểu được kiến trúc cơ bản của Amazon EC2 và các thành phần chính của nó:
  * Instance Types
  * Amazon Machine Images (AMI)
  * Elastic Block Store (EBS)
  * Security Groups
  * Key Pairs

* Khởi chạy thành công một EC2 instance và thiết lập kết nối SSH bằng key pair đã tạo.

* Có kinh nghiệm thực tế sử dụng cả AWS Management Console và AWS CLI để thực hiện các tác vụ quản trị giống nhau.

* Xây dựng nền tảng vững chắc để học các dịch vụ AWS nâng cao hơn về mạng, lưu trữ và triển khai ứng dụng trong các tuần tiếp theo.
