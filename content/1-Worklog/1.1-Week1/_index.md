### Week 1 Achievements:

* Understood the fundamental concepts of cloud computing and the advantages of using cloud services over traditional infrastructure.

* Learned the AWS global infrastructure, including:
  * Regions
  * Availability Zones
  * Edge Locations
  * Global Services

* Successfully created and secured an AWS Free Tier account for hands-on practice.

* Became familiar with the AWS Management Console and understood how AWS services are organized.

* Learned the purpose of AWS Identity and Access Management (IAM), including:
  * IAM Users
  * IAM Groups
  * IAM Policies
  * Least Privilege Principle

* Installed and configured AWS CLI successfully, including:
  * Access Key ID
  * Secret Access Key
  * Default Region
  * Output Format

* Used AWS CLI to perform basic management tasks such as:
  * Verify CLI configuration
  * Retrieve available AWS regions
  * List IAM users
  * View account configuration
  * Explore available AWS services

* Understood the basic architecture of Amazon EC2 and its main components:
  * Instance Types
  * Amazon Machine Images (AMI)
  * Elastic Block Store (EBS)
  * Security Groups
  * Key Pairs

* Successfully launched an EC2 instance and established an SSH connection using a generated key pair.

* Gained practical experience using both the AWS Management Console and AWS CLI to perform the same administrative tasks.

* Built a solid foundation for learning more advanced AWS networking, storage, and application deployment services in the following weeks.
