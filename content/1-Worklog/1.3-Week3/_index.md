### Week 3 Achievements:

* Understood the architecture and use cases of Amazon Simple Storage Service (S3).

* Learned the core concepts of Amazon S3, including:
  * Buckets
  * Objects
  * Storage Classes
  * Bucket Policies
  * Static Website Hosting

* Successfully created and configured an S3 bucket to host a static website.

* Learned how to manage object permissions and enable public access securely.

* Understood the purpose of Amazon Route 53 and its role as a managed DNS service.

* Successfully configured Route 53 components, including:
  * Hosted Zones
  * A Records
  * Alias Records
  * DNS Resolution

* Learned how Amazon CloudFront distributes content globally through edge locations.

* Successfully created a CloudFront distribution for the static website and verified content delivery through the CDN.

* Understood the concepts of:
  * Cache Behavior
  * Cache Invalidation
  * Origin Configuration
  * HTTPS Content Delivery

* Successfully integrated Amazon S3, Route 53, and CloudFront to build a secure and highly available static website.

* Gained practical experience in deploying static web applications using AWS managed services.

* Established a strong foundation for hosting scalable web applications and preparing for database integration in the following weeks.
