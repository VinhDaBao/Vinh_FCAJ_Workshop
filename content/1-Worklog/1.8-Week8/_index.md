### Week 8 Objectives:

* Understand the fundamentals of containerization and Docker.
* Learn how to deploy containerized applications using Amazon ECS.
* Explore container image management with Amazon ECR.
* Gain hands-on experience deploying scalable applications using AWS container services.

### Week 8 Achievements:

* Understood the concepts and advantages of containerization compared to traditional virtual machines.

* Learned the core components of Docker, including:
  * Docker Images
  * Docker Containers
  * Dockerfile
  * Docker Hub
  * Docker Engine

* Successfully built, executed, and managed Docker containers on a local development environment.

* Learned how Amazon Elastic Container Registry (ECR) stores and manages container images securely.

* Successfully pushed and managed Docker images in Amazon ECR.

* Understood the architecture of Amazon Elastic Container Service (ECS), including:
  * ECS Clusters
  * Task Definitions
  * ECS Services
  * Fargate Launch Type
  * EC2 Launch Type

* Successfully deployed a containerized application on Amazon ECS using AWS Fargate.

* Learned how ECS automatically manages container scheduling, scaling, and availability.

* Successfully updated application containers through rolling deployments with minimal service interruption.

* Monitored ECS tasks and services using Amazon CloudWatch to verify application health and performance.

* Gained practical experience deploying cloud-native applications using Docker, Amazon ECR, and Amazon ECS.

* Built a strong foundation for implementing CI/CD pipelines and automated application deployment in the following weeks.
