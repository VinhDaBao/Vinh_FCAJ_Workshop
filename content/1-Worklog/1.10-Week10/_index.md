### Week 10 Achievements:

* Understood the AWS Shared Responsibility Model and the importance of implementing security throughout the cloud infrastructure.

* Learned AWS security best practices, including:
  * Identity and Access Management
  * Least Privilege Principle
  * Multi-Factor Authentication (MFA)
  * Resource Access Control

* Successfully implemented encryption using AWS Key Management Service (KMS) to protect sensitive application data.

* Learned how AWS Secrets Manager securely stores and manages application credentials and API keys.

* Successfully configured encrypted secrets and understood automatic secret rotation.

* Understood how AWS WAF protects web applications from common web attacks by filtering HTTP and HTTPS requests.

* Learned the capabilities of AWS Shield for mitigating Distributed Denial-of-Service (DDoS) attacks.

* Enabled Amazon GuardDuty to continuously monitor AWS accounts for suspicious activities and potential security threats.

* Used AWS Security Hub to aggregate security findings and review compliance across AWS services.

* Gained practical experience identifying security vulnerabilities and applying appropriate remediation measures.

* Improved understanding of securing production-ready cloud environments using AWS managed security services.

* Established a strong security foundation for designing reliable, compliant, and enterprise-grade cloud architectures.
