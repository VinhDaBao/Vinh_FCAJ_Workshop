This worklog summarizes my 12-week learning journey in the **First Cloud AI Journey (FCAJ)** program. Throughout the internship, I studied core AWS services, completed hands-on laboratories, and gradually built practical cloud computing skills.

The learning roadmap was organized progressively, beginning with the fundamentals of AWS and continuing through networking, storage, databases, serverless computing, containers, DevOps, security, monitoring, cost optimization, and cloud architecture. Each week consisted of learning objectives, practical exercises, and a summary of the knowledge and skills acquired.

The following sections present the weekly activities completed during the internship.

**Week 1:** [Introduction to AWS, AWS Account, IAM, AWS CLI, and Amazon EC2](1.1-week1/)

**Week 2:** [Amazon VPC, Subnets, Route Tables, Security Groups, Network ACLs, and Elastic IP](1.2-week2/)

**Week 3:** [Amazon S3, Static Website Hosting, Amazon Route 53, and Amazon CloudFront](1.3-week3/)

**Week 4:** [Amazon RDS, Amazon DynamoDB, and Amazon ElastiCache](1.4-week4/)

**Week 5:** [Amazon CloudWatch, Elastic Load Balancing, and Auto Scaling](1.5-week5/)

**Week 6:** [AWS Lambda, Amazon API Gateway, Amazon SQS, and Amazon SNS](1.6-week6/)

**Week 7:** [AWS CloudFormation, AWS CDK, and AWS Systems Manager](1.7-week7/)

**Week 8:** [Docker, Amazon ECR, and Amazon ECS](1.8-week8/)

**Week 9:** [AWS CodeCommit, CodeBuild, CodeDeploy, and CodePipeline (CI/CD)](1.9-week9/)

**Week 10:** [AWS Security Services: IAM Best Practices, KMS, Secrets Manager, WAF, GuardDuty, and Security Hub](1.10-week10/)

**Week 11:** [AWS Backup, Cost Explorer, AWS Budgets, Trusted Advisor, and Cost Optimization](1.11-week11/)

**Week 12:** [Final Cloud Solution Deployment, Architecture Review, and Program Summary](1.12-week12/)
