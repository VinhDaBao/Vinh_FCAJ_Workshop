### Week 7 Achievements:

* Understood the principles and advantages of Infrastructure as Code (IaC) for automating AWS resource provisioning.

* Learned the core components of AWS CloudFormation, including:
  * Templates
  * Stacks
  * Stack Updates
  * Change Sets

* Successfully created and deployed AWS resources using CloudFormation templates.

* Learned how AWS Cloud Development Kit (CDK) enables infrastructure deployment through programming languages.

* Understood the AWS CDK workflow, including:
  * Construct creation
  * Stack synthesis
  * Deployment
  * Resource management

* Successfully deployed cloud resources using AWS CDK and verified infrastructure creation.

* Learned the capabilities of AWS Systems Manager, including:
  * Session Manager
  * Parameter Store
  * Run Command

* Successfully connected to EC2 instances using Session Manager without requiring SSH access.

* Stored application configuration securely using Parameter Store and retrieved values for application deployment.

* Executed remote administrative commands on EC2 instances using Systems Manager Run Command.

* Gained practical experience automating infrastructure deployment and system management using AWS native services.

* Built a solid foundation for containerization and cloud-native application deployment in the following weeks.
